RelatedContent
==============

RelatedContent adds to each node a tab from which nodes to be asociated with
the node, and optionlly displayed along with the it, can easily be selected by
privileged users. Fore more information, please read the on-line documentation
at http://drupal.org/node/130258.


Requirements
------------

To install RelatedContent you need:

* Drupal 6.x
* Views module


Installation
------------

Install RelatedContent as follows:

  1. Download, install and configure the Views module, following the
     instructions for that module.
  2. Download the latest stable version of RelatedContent from its project
     page.
  3. Unpack the downloaded file into sites/all/modules or the modules directory
     of your site.
  4. Go to Administer » Site building » Modules and enable the module.


Configuration, usage, theming
-----------------------------

For information about configuration, usage and theming of this module, please
see the online help provided by the module. Go to Administer » Help, and click
on the RelatedContent link.

This information is also aviable on-line at http://drupal.org/node/130258.
